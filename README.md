# LR1 Parser Project

![Recognize Dyck words](/images/dyck.png)

## Project in C
Usage:

```
make
./automaton path_to_aut_file
```
:warning: :warning:  :warning: 
path_to_aut_file must correspond to a correctly written .aut file, otherwise the program won't work
